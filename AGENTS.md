# Global Agent Configuration

## Terminal Environment
- Shell: zsh
- Terminal Emulator: Kitty
- Platform: macOS (darwin)

## Installed Tools
- **fzf** (v0.67.0) - Fuzzy finder for files and command history
  - Ctrl+R: Search command history
  - Ctrl+T: Search files in current directory
  - Alt+C: Change directory using fuzzy search
- **zoxide** (v0.9.8) - Smart directory jumping tool
  - `z <directory>`: Jump to frequently used directory
  - `zi <directory>`: Interactive selection with fzf integration
- **Neovim** (v0.11.6) - Hyperextensible Vim-based text editor

## Instructions
When starting a session, you already know the terminal environment without needing to run commands to detect it. Use the terminal environment information provided above.
